# MirageAI 使用指南

## 🔧 配置说明

### 1. API Key 配置

将你的阿里云 DashScope API Key 添加到 `.env.local` 文件：

```bash
# 复制示例文件
cp .env.local.example .env.local

# 编辑文件，添加你的API Key
DASHSCOPE_API_KEY=your_actual_api_key_here
```

### 2. 获取 API Key

1. 访问 [阿里云百炼控制台](https://bailian.console.aliyun.com/)
2. 登录阿里云账号
3. 创建 API Key
4. 将 API Key 复制到 `.env.local` 文件中

## 🚀 启动应用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

## 📝 使���流程

### 第一步：故事输入
1. 在文本框中输入你的故事梗概
   - 例如："一个少年在雨夜中奔跑，寻找失踪的妹妹"
2. 选择视频参数：
   - 视频比例：16:9（横屏）或 9:16（竖屏）
   - 视频分辨率：480P（默认）或 720P
   - 画面风格：赛博朋克、皮克斯、恐怖、写实、国风、动漫
3. 点击"下一步"按钮

### 第二步：分镜头编辑
1. 查看AI生成的4个分镜头脚本
2. 查看根据主角描述词生成的主角图片
3. 可以编辑每个分镜头的描述文字
4. 点击"生成视频"按钮

### 第三步：视频预览
1. 预览生成的4个分镜头视频片段
2. 点击缩略图可以切换查看不同镜头
3. 确认无误后点击"生成最终视频"

### 第四步：完成下载
1. 查看合并后的完整视频
2. 点击"下载视频"保存到本地
3. 可以分享作品或创建新视频

## ⚠️ 注意事项

### API 使用限制
- 确保你的阿里云账户有足够的余额
- API 调用有频率限制，请合理安排使用
- 生成的视频和图片链接有效期为24小时

### 生成时间
- 分镜头生成：约2-3秒
- 主角图片生成：约10-30秒
- 每个视频片段生成：约2-5分钟
- 总生成时间：约10-20分钟

### 浏览器兼容性
- 推荐使用最新版本的 Chrome、Firefox、Safari 或 Edge
- 确保浏览器支持 HTML5 视频播放

## 🐛 故障排除

### 问题1：API调用失败
- 检查 `.env.local` 文件中的 API Key 是否正确
- 确认网络连接正常
- 检查阿里云账户余额

### 问题2：视频生成失败
- 检查分镜头描述是否合适
- 尝试使用不同的风格
- 确认主角图片生成成功

### 问题3：下载失败
- 尝试右键视频选择"另存为"
- 检查浏览器下载设置
- 清除浏览器缓存后重试

## 📱 移动端使用

应用完全支持移动端浏览器：
- 响应式设计自适应屏幕
- 触摸友好的交互
- 支持横屏和竖屏视频

## 💡 使用建议

1. **故事梗概技巧**：
   - 描述要简洁明了
   - 包含时间、地点、人物、事件
   - 控制在50-200字之间

2. **分镜头编辑**：
   - 保持每个镜头的独立性
   - 确保镜头之间的连贯性
   - 可以添加动作和情感描述

3. **风格选择**：
   - 赛博朋克：适合科幻、未来主题
   - 皮克斯：适合温馨、童话风格
   - 恐怖：适合悬疑、惊悚故事
   - 写实：适合现实题材
   - 国风：适合古典、武侠题材
   - 动漫：适合二次元风格

## 🆘 技术支持

如果遇到问题：
1. 查看浏览器控制台错误信息
2. 检查网络连接
3. 确认 API 配置正确
4. 重新启动应用
